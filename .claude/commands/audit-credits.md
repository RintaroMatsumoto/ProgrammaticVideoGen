# /audit-credits [slug?]

すべての浮世絵作品、または指定した slug の `public/ukiyoe/<slug>/CREDITS.md` を点検する。

## チェック項目

動画を公開する前に、以下がすべて揃っているか確認する。抜けがあれば具体的に指摘し、修正案を示す。

### 画像
- [ ] 作者名（没年つき）
- [ ] 作品名（JP / EN）
- [ ] 制作年
- [ ] PD の根拠（作者没年 or 発表から経過年数）

### ナレーション
- [ ] VOICEVOX の表記
- [ ] 話者名（例：九州そら）
- [ ] 商用利用許諾の記載

### BGM
- [ ] 楽曲名・作曲者
- [ ] 演奏者 / 録音
- [ ] 出所 URL
- [ ] ライセンス種別（PD / CC0 / CC BY / CC BY-NC など）
- [ ] ライセンスが **CC BY 系の場合は帰属文字列の完全版**（`docs/CONVENTIONS.md` §5 参照）
- [ ] 処理の記載（trim / normalize / denoise など）

### バックアップ
- [ ] 別バージョンの BGM を温存している場合はその旨記載

## 出力

各項目ごとに `OK` / `MISSING` / `UNCLEAR` を返し、問題があれば修正すべき行を具体的に示す。

公開差し止めレベルの欠落（特に CC BY 帰属の抜け）は冒頭で **「公開不可」** を明記する。
