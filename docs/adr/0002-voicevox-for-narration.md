# ADR-0002: ナレーションに VOICEVOX を採用

- Date: 2026-04-16
- Status: accepted

## Context

ナレーション音声の選択肢：

- **クラウド TTS（Azure / Google / ElevenLabs）**：高品質だが従量課金、音声の個性が弱い
- **VOICEVOX**：無料・ローカル・商用可、キャラクター制約（クレジット表記必要）
- **CoeFont / OpenAI TTS**：利用規約要確認、品質高いが商用条件が流動的
- **人力収録**：最高品質、最高コスト

## Decision

**VOICEVOX** を採用。話者は `16: 九州そら（ノーマル）` を既定とする。

## Consequences

良い面：
- ローカルで完結（API コスト 0 円）
- 商用利用可（キャラクタークレジットのみ）
- 落ち着いた女性声が浮世絵解説に合う
- シーンごとに話者を切り替え可能（scene.speaker 属性）

悪い面：
- エンジン起動（`voicevox_engine/run.exe`）が前提。忘れると全滅
- 発音辞書が貧弱な固有名詞（人名・地名）では読みが崩れる
  → 事前に `narration_ja` を ひらがな置換する余地を残すこと
- 感情表現は淡白。クラウド勢のように「熱く語る」は苦手

トレードオフ：
- 品質は ElevenLabs に劣るが、浮世絵解説という「静かな語り」ジャンルには過不足なし
- ランニングコスト 0 は事業の持続性に大きく効く
